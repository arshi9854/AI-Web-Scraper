# 🕸️ AI Web Scraper

An intelligent, automated **AI-powered web scraper** built with **Ollama**, **Bright Data**, **Selenium**, and other powerful libraries. This project is designed to navigate, extract, and analyze web content intelligently — even on sites with CAPTCHAs and dynamic loading.

---

## 🚀 Features

- 🌐 **Proxy support** via [Bright Data](https://brightdata.com) for scalable, global access
- 🤖 **Ollama integration** for smart content analysis and language understanding
- 🧠 **CAPTCHA handling-ready** (if Bright Data Scraping Browser is enabled)
- 🧭 **Selenium automation** for full browser control
- 📄 **HTML to text extraction** for cleaner content parsing
- 🛠️ Modular codebase for easy customization

---

## 🧰 Tech Stack

- 🦾 [Ollama](https://ollama.com/) – local LLM runtime for AI tasks
- 🌐 [Bright Data](https://brightdata.com/) – proxy network and web scraping tools
- 🧪 [Selenium](https://www.selenium.dev/) – browser automation
- 🐍 Python 3.8+
- 📦 Streamlit – for interactive UI (optional)
